Instuctions Voice Assistant

Command Switch to sleep mode”, “Sleep Mode" - puts the pc into sleep mode;

Command "Turn off the computer", "Shutdown Mac" - turns off your pc;

Command "Please turn up the volume" - increases the volume by 10 points;

Command "Please turn down the volume" - decreases the volume by 10 points;

Command "Create note [note content]" - to create new notes;

Command "Open website: [website name]" - to open a website;

Command "What time is it?", "Current time","What time is it now?" - tells the current tense;

Command "What date is today?", "Can you name the current date today?" - tells the current date;
